# FastAPI CRUD Application

This repository features a complete, production-ready CRUD (Create, Read, Update, Delete) API built using FastAPI and SQLAlchemy. 

## The Development Journey
* **`main.py` (Tutorial-Guided):** Initially built by following a YouTube tutorial to establish the baseline configuration and understand basic routing principles.
* **Branched Version (Independent Build):** Built entirely from scratch without any tutorials or external guides.  Forcing myself to do this allowed me to master the intersection of database sessions, Pydantic data serialization, and full RESTful CRUD architectures.

##  Features
* Full CRUD endpoints for user management.
* SQLite database integration via SQLAlchemy ORM.
* Strict request/response data validation utilizing Pydantic schemas.
* Built-in interactive documentation via Swagger UI (`/docs`).
